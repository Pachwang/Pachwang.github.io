实验一

1.查看虚拟机的 IP 信息

2.利用SSH客户端（如SecureCRT/XShell）进行虚拟机远程连接。以下内容在SSH客户端上完成

3.测试虚拟机与FTP服务器(172.26.14.30)的连通性(pig172.26.14.30)。通过ctr1+c组合键强行退出正在运行的程序

4.安装上传下载工具Irzsz

5.查看Irzsz工具中rz和sz的路径

6.在Window/MacOS下创建一个名为实验一.txt的文件，并在文件中写入Linux第一次实验，然后上传到linux中

7.把实验一.txt移动到/tmp目录下，并重命名为exam1.txt

8.把exam1.txt转成unix格式（命令未安装请自行安装）

9.利用vi打开exam1.txt(可能有中文乱码)，并在第一行插入自己的学号，然后保存退出

10.利用重定向把字符串1234567890追加到exam1.txt的末尾

11.把/etc/passwdi的最后5行追加到exam1.txt中

12.搜索/usr下所有以xml结尾的文件（只搜索普通文件），并把路径中含有mysql的文件路径追加到exam1.txt中

13.把当前时间按照年-月-日时:分:秒的格式追加到exam1.txt中。如：2020-11-23 09:32:43

14.对目录/var/log进行压缩生成名为log.tar.gz文件

15.通过ls命令以长格式的形式查看log.tar.gz的信息，并把信息追加到exam1.txt中

16.利用awk命令提取上一步中打印的1og.tar.gz的文件类型和权限信息追加到exam1.txt中

17.把exam1.txt转换成window格式

18.把exam1.txt传递到window下

19.把exam1.txt重命名为学号.txt然后提交

实验二

1.利用SSH客户端登录root账号，查看/tmp目录下是否存在子目录myshare,如果没有则建立该目录

2.在myshare目录下创建一个名为”学号”的文件夹和一个名为exam2.txt的文件

3.创建一个名字为test的新用户，并指定uid为1024

4.把/etc/passwd和/etc/shadow含有用户test信息的行追加到exam2.txt文件中

5.把/etc/passwd前13行的内容追加到myshare目录下名为exam2.txt的文件中

6.把myshare目录下的所有文件和子目录的内容以长格式的方式追加到exam2.txt中

7.把myshare目录及其目录下的所有文件和子目录的拥有者设置为用户test,组改为mail

8.把myshare目录下的所有文件和子目录的内容以长格式的方式追加到exam2.txt中

9.利用su命令切换到用户test账号

进入/tmp/myshare/“学号”目录，采用vi编辑器编写以下程序，程序名称为hello.sh:

10.保存hello.sh后，给予hello.sh拥有者可读、可写和可执行的权限，同组可读可执行，其他人可执行权限

11.以长格式的形式查看hello.sh的文件权限信息，并把输出内容追加到exam2.txt

12.输入./hello.sh执行脚本，查看输出结果。并把输出结果追加exam2.txt

13.进入用户test的用户主目录，在这个目录下创建hello.sh的软链接myhello.sh,同时拷贝hello.sh到该目

录下并改名为hello.sh.bak

14.以长格式形式查看用户test主目录下的所有文件并把结果追加到exam2.txt中

15.执行用户test主目录下的myhello.sh文件，查看链接是否正常

16.退出用户test帐号，回到root帐号

17.以长格式形式查看用户test主目录下的所有文件含隐藏文件)并把结果追加到exam2.txt中

18.从/开始查找后缀名为.conf的所有文件（普通文件），把输出结果追加到exam2.txt中

19.从上一步找到的con文件中我出文件容量最大的文件，井把这个文件以长格式形式追加到exam2.txt中：（倒引号】[通过命令方式实现]

20.统计出系统中有多少个用户帐号，把数量追加到exam2.txt中

21.把exam2.txt文件转换为windows格式

22.把exam2.txt传送到window下

23.删除⽤户test 的所有内容（包括主⽬录)

24.删除/tmp/myshare⽬录

25.把 exam2.txt 重命名为 学号.txt 然后提交
